## 📁 Migration Governance & Logic (CSC/DXC Era)

### **1. The "6-R" Migration Strategy**
* **Rehost (Lift & Shift):** Optimized ASR (Azure Site Recovery) workflows for rapid movement of legacy Windows/Linux VMs.
* **Replatform:** Upgraded OS levels and storage configurations during the migration window to ensure cloud compatibility.

### **2. Cloud Connectivity & Security**
* **Network Segmentation:** Designed Azure Virtual Networks (VNETs) and Subnets to mirror on-premises security zones (DMZ/Internal).
* **NSG Governance:** Implemented Network Security Groups (NSGs) as a first line of defense for migrated workloads.

### **3. Post-Migration Optimization**
* **Right-Sizing:** Utilized Azure Monitor data to resize VM instances, reducing cloud consumption costs by 20% post-migration.
* **Backup & DR:** Standardized Azure Backup vaults and ASR replication policies for all migrated Tier-1 assets.
